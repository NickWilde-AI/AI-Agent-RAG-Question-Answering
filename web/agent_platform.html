Agent Runtime 执行链路

中台的核心不是堆功能，而是把模型能力变成可靠、可控、可审计的执行系统。

Harness
1. Intent识别用户任务：问答、审批、分析、控制、研究。
2. PlanPlan-and-Execute / LangGraph 拆解任务步骤。
3. Route选择 Agent、Skill、MCP 工具和模型 Provider。
4. Execute调用 RAG、VLM、数据库、审批、设备控制。
5. Verify证据校验、动作校验、失败重试和降级。
6. TraceSSE 轨迹、审计、评测、成本和复盘。
