# private / 本地隐私区

本目录用于存放**个人简历、面试笔记、宣讲 PDF、从 PDF 抽取的文本**等仅在本机使用的材料。

- **不要**将本目录下除本说明外的文件提交到公开仓库（根目录 `.gitignore` 已忽略 `private/*`，并保留 `!private/README.md`）。
- 开源协作与对外说明以仓库根目录 **`README.md`** 与 **`.env.example`** 为准。

建议将以下类型文件统一放在此目录（可按需建子目录，如 `pdf_extract/`）：

- 简历 Markdown / PDF  
- 面试逐字稿、技术对照表  
- `RAG 项目总结` 等个人复习笔记  
- 从简历 PDF 抽取的 `.txt` 等
